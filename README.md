# Lab 6 – AI Agent Integration

This project extends the Lab 5 system by introducing an AI agent capable of reasoning over user requests and executing tools to generate responses.

The system integrates data retrieval, analytics, and an interactive interface where users can ask questions and receive responses generated by the agent.

---

## System Overview

The system consists of four main components:

• AI Agent responsible for reasoning and tool selection
• Tools that perform operations such as analytics or data retrieval
• Streamlit interface that allows users to interact with the system
• Dataset used for analysis and responses

The agent receives user input, determines which tools are required, executes them, and produces a final response.

---

## Agent

The agent interprets user queries and determines which tools should be used.

The agent supports multi-step reasoning and may call multiple tools before generating the final response.

File:
agent.py

---

## Tools

The project defines tools that the agent can call.

Each tool is implemented as a Python function with:

• clear docstrings
• type hints
• error handling

Files:

tools.py
tool_schemas.py

---

## Streamlit Interface

The system includes an interactive Streamlit interface where users can ask questions and view responses from the agent.

Features include:

• chat-based interface
• conversation history
• loading indicator while the agent processes a request

File:

streamlit_app.py

---

## Dataset

The project uses a dataset containing Toyota stock price data from 1980 to 2026.

File:

Toyota_Stock_Prices_1980_2026.csv

The dataset is used for simple analytics and demonstrations of tool usage.

---

## Running the Project

Install dependencies:

pip install -r requirements.txt

Run the Streamlit application:

streamlit run streamlit_app.py
